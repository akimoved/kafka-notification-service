# Система аутентификации с подтверждением по email

Два микросервиса на Spring Boot с Kafka интеграцией.

## Структура проекта

```
auth-service/          # Сервис аутентификации (порт 8080)
notification-service/  # Сервис рассылки (порт 8081)
```
[Репозиторий Authentication Service и README](https://github.com/akimoved/kafka-auth-service)
